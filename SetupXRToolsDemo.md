# Setup XR Tools Demo

## Overview

Setup a Godot XR tools demo project.

## Instructions

1. Get the Godot version

   ```bash
   godot --version
   ```

2. Get the compatible Godot XR Tools version for the current Godot version

   Version compatibility:

   | Godot XR Tools | Minimum Godot version |
   | -------------- | --------------------- |
   | 4.5.x          | 4.4                   |
   | 4.4.x          | 4.2                   |
   | 4.3.x          | 4.1                   |
   | 4.2.x          | 4.1                   |
   | 4.1.x          | 4.0                   |
   | 4.0.x          | 4.0                   |
